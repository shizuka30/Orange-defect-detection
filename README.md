# 🍊 Orange Defect Detection using Deep Learning

## 📌 Overview
This project focuses on detecting defects in oranges using deep learning. 
The model classifies oranges into multiple categories such as:
- Fresh
- Blackspot
- Canker
- Greening

This can be used in agriculture and food quality inspection systems to automate defect detection.

---

## 🧠 Problem Statement
Manual inspection of fruits is time-consuming and error-prone.  
This project aims to build an AI model that can automatically identify defects in oranges using image data.

---

## ⚙️ Approach
- Image classification using Convolutional Neural Networks (CNN)
- Dataset split into training and testing sets
- Data preprocessing and normalization
- Model training and evaluation

---

## 📊 Model Details
- Model: CNN
- Loss Function: Categorical Crossentropy
- Optimizer: Adam
- Evaluation Metric: Accuracy

---
